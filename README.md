# Deep Research AI Assistant

An AI-powered Multi-Agent Research System that performs deep research on any topic, generates a comprehensive research report, and provides a downloadable PDF version of the report.

Built using the OpenAI Agents SDK, Gradio, XHTML2PDF, AsyncIO, and Pydantic.

---

## Overview

Deep Research AI Assistant is a multi-agent application that automates the research process by dividing the task among specialized AI agents. Instead of relying on a single agent, the system follows an agentic workflow where each agent is responsible for a specific task such as planning, information gathering, report generation, and document formatting.

The application accepts a research topic from the user, performs multiple targeted searches, synthesizes the findings into a structured report, converts the report into HTML, generates a PDF document, and finally presents both the report and PDF through a Gradio-based user interface.

## Architecture

### 1. Planner Agent

The Planner Agent analyzes the user's research topic and creates multiple focused search queries.

Responsibilities:

* Understand the user's research intent
* Generate multiple search queries
* Provide reasoning for each query
* Create a structured research plan

---

### 2. Search Agent

The Search Agent executes the search plan generated by the Planner Agent.

Responsibilities:

* Perform web research
* Gather information from multiple sources
* Return summarized search results

The searches are executed concurrently using AsyncIO to improve performance.

---

### 3. Research Agent

The Research Agent processes all collected search results and generates a detailed research report.

Responsibilities:

* Analyze gathered information
* Identify important findings
* Remove redundant content
* Produce a coherent research report

---

### 4. HTML Converter Agent

The HTML Converter Agent transforms the generated report into well-structured HTML.

Responsibilities:

* Convert report content to HTML
* Apply document structure
* Prepare content for PDF generation

---

### 5. PDF Generation Module

The PDF generation module converts the HTML report into a downloadable PDF document.

Technologies Used:

* XHTML2PDF
* Temporary File Management

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/deep-research-ai-assistant.git
cd deep-research-ai-assistant
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Virtual Environment

Windows:

```bash
.venv\Scripts\activate
```

Linux / Mac:

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```
